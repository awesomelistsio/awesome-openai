> A curated list of OpenAI tools, APIs, research, SDKs, and community resources.

OpenAI is an AI research and deployment company behind models like GPT, DALL·E, and Codex. This list curates tools, resources, and projects that help developers, researchers, and enthusiasts work with OpenAI technologies.

## Official Resources

- **[OpenAI](https://openai.com/)** – The official website of OpenAI.
- **[OpenAI Platform](https://platform.openai.com/)** – API access and documentation.
- **[OpenAI Documentation](https://platform.openai.com/docs)** – Official developer documentation.
- **[OpenAI Status](https://status.openai.com/)** – Service status and incident reports.
- **[OpenAI GitHub](https://github.com/openai)** – Official open-source projects.

## APIs & SDKs

- **[OpenAI Node.js SDK](https://github.com/openai/openai-node)** – Official Node.js library for OpenAI APIs.
- **[OpenAI Python SDK](https://github.com/openai/openai-python)** – Official Python library for accessing OpenAI’s API.
- **[OpenAI Ruby SDK](https://github.com/alexrudall/ruby-openai)** – Ruby gem to interact with OpenAI APIs.

## Third-Party Libraries

- **[LangChain](https://github.com/langchain-ai/langchain)** – Framework for building LLM apps with memory, tools, and agents.
- **[LlamaIndex](https://github.com/jerryjliu/llama_index)** – LLM data framework for connecting language models to external data.
- **[Auto-GPT](https://github.com/Torantulino/Auto-GPT)** – Experimental open-source agent with OpenAI and automation.
- **[GPT-Engineer](https://github.com/AntonOsika/gpt-engineer)** – Specify what you want and let GPT-4 build the codebase.

## ChatGPT

- **[ChatGPT](https://chat.openai.com/)** – OpenAI’s web interface for conversational AI.
- **[ChatGPT Plugins](https://platform.openai.com/docs/plugins/introduction)** – Extend ChatGPT with external tools and APIs.
- **[ChatGPT API](https://platform.openai.com/docs/guides/gpt)** – Use GPT-4 and GPT-3.5 models via API.
- **[ChatGPT Prompt Engineering Guide](https://github.com/dair-ai/Prompt-Engineering-Guide)** – Best practices for designing effective prompts.

## DALL·E

- **[DALL·E 3](https://openai.com/dall-e)** – Text-to-image model with inpainting and prompt enhancements.
- **[OpenAI Image API](https://platform.openai.com/docs/guides/images)** – Generate and edit images via API.
- **[DALL·E Playground](https://huggingface.co/spaces/dalle-mini/dalle-mini)** – Community tools and UIs for experimentation.

## Whisper

- **[Whisper](https://github.com/openai/whisper)** – OpenAI’s automatic speech recognition system.
- **[Whisper API](https://platform.openai.com/docs/guides/speech-to-text)** – Convert audio into text via API.
- **[WhisperX](https://github.com/m-bain/whisperX)** – Fast automatic speech recognition with word-level timestamps.

## Codex & Code Models

- **[OpenAI Codex](https://openai.com/blog/openai-codex)** – AI system that translates natural language into code.
- **[Code Interpreter / Advanced Data Analysis](https://openai.com/blog/chatgpt-plugins#code-interpreter)** – Execute code and analyze files inside ChatGPT.
- **[GitHub Copilot](https://github.com/features/copilot)** – AI pair programmer powered by Codex, developed with GitHub.

## Learning Resources

- **[OpenAI Cookbook](https://github.com/openai/openai-cookbook)** – Examples and guides for working with the OpenAI API.
- **[Prompt Engineering Guide](https://github.com/dair-ai/Prompt-Engineering-Guide)** – In-depth resource for crafting prompts.
- **[Full Stack GPT](https://github.com/AIAnytime/full-stack-gpt)** – Tutorials for building full-stack applications with OpenAI.
- **[Build with OpenAI (YouTube)](https://www.youtube.com/@OpenAI)** – Official video tutorials and demo walkthroughs.

## Research & Papers

- **[GPT-4 Technical Report](https://openai.com/research/gpt-4)** – Overview of OpenAI’s multimodal GPT-4 model.
- **[Language Models are Few-Shot Learners (GPT-3)](https://arxiv.org/abs/2005.14165)** – Seminal paper introducing GPT-3.
- **[Whisper Paper](https://cdn.openai.com/papers/whisper.pdf)** – Research report on OpenAI’s ASR system.

## Projects & Demos
